First feedback gotten

During the first pitch day on the 13th of november, I received feedback from 3 students (Lars, Serghei and Jaime) and a teachers assistant (Timo). The main feedback I got were: Add meta tags, select a general colour palette, sort the CSS in multiple files, add a footer in the dashboard and put the blog links in little sections. I chose to implement most feedback, but I chose not to implement the multiple CSS files strategy since you can simply search for a specific ID or class in the css file using CTRL+F.
